<h1 align="center"><b>Toolbox for MacOS</b></h1>

Features

- simple window manager
- caffeinate

Configuration file must be placed at one of the following places (in order):

- `$HOME/.config/toolbox/config.yml`
- `$HOME/.toolbox.yml`

Sample:

```yml
window_padding: 10.0
window_gap: 10.0
border_width: 2.0
border_radius: 10.0
accent_color: '#44fcee'
```
